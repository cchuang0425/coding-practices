 ---- | ---- | ----

HackerRank | HR | [HackerRank](https://www.hackerrank.com/)
UVa OnlineJudge | OJ | [UVa OnlineJudge](https://uva.onlinejudge.org/index.php)

如上表所列，repo 的目錄規則為 `{prefix}-{題號}/{lang}`，以便整理。
